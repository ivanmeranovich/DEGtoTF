# DEGtoTF

Скрипт для сопоставления данных дифференциальной экспрессии генов (DEG) с таблицей транскрипционных факторов (TF) с последующей фильтрацией по статистическим критериям.

---

## 📌 Описание

`DEGtoTF.py` объединяет одну таблицу транскрипционных факторов (TF) с одной или несколькими таблицами DEG по ключу:

```
Locus Tag (TF) == locus (DEG)
```

После сопоставления значения `logFC` и `FDR` из DEG добавляются к TF и проводится фильтрация по заданным порогам.

---

## ⚙️ Требования

* Python 3.8+
* Установленные библиотеки:

  ```bash
  pip install pandas openpyxl
  ```

---

## 📂 Входные данные

### 1. Таблица TF (обязательная, одна)

Обязательный столбец:

* `Locus Tag`

Полная структура:

```
Index Number 
Protein ID 
Gene 
Locus Tag 
Critical Residue Tag 
Family 
Accession 
E-value 
Score 
Description
```

---

### 2. Таблицы DEG (одна или несколько)

Обязательные столбцы:

* `locus`
* `logFC`
* `FDR`

Полная структура:

```
target_id 
locus 
gene 
product 
logFC 
logCPM 
PValue 
FDR
```

---

## 🚀 Использование

```bash
python DEGtoTF.py -TF <tf_file> -DEG <deg1> [-DEG <deg2> ...] -OUT <output.xlsx> -LogFC <float> -FDR <float>
```

### Аргументы:

| Параметр | Описание                                   |
| -------- | ------------------------------------------ |
| `-TF`    | Таблица транскрипционных факторов (1 файл) |
| `-DEG`   | Таблица DEG (можно указать несколько раз)  |
| `-OUT`   | Имя выходного файла (.xlsx)                |
| `-LogFC` | Порог по модулю для logFC                  |
| `-FDR`   | Порог для FDR                              |

---

### Пример:

```bash
python DEGtoTF.py \
  -TF tf.xlsx \
  -DEG deg1.xlsx \
  -DEG deg2.xlsx \
  -OUT result.xlsx \
  -LogFC 1.5 \
  -FDR 0.05
```

---

## 🔄 Логика работы

### 1. Сопоставление (Mapping)

* Загружается TF таблица
* Загружаются все DEG таблицы
* Выполняется объединение (`merge`) по:

  ```
  Locus Tag == locus
  ```
* Для каждой DEG таблицы добавляются столбцы:

  * `LogFC1`, `FDR1`
  * `LogFC2`, `FDR2`
  * и т.д.

---

### 2. Фильтрация

Строка сохраняется **только если выполняются условия для ВСЕХ DEG таблиц**:

```
FDR <= FDR_threshold
AND
abs(LogFC) >= LogFC_threshold
```

---

### ❗ Важно

* Если для TF отсутствуют данные хотя бы в одной DEG таблице → строка удаляется
* Если хотя бы в одной DEG таблице не выполняется условие → строка удаляется

## 🖥️ Вывод в консоль

Скрипт выводит:

1. Количество TF, для которых найдено совпадение с DEG
2. Количество TF, прошедших фильтрацию

Пример:

```
Сопоставление завершено: найдено совпадений для 120 TF
После фильтрации осталось 35 TF со статистически значимыми значениями
Результат сохранён в файл: result.xlsx
```
